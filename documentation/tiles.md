![](reference-alt.png)

- `0z`, `8z`, `9z`, `00s`, `10m`, `10p`, `10s`
- `1x`, and a bunch of spaces:
  + `2x` (tile-width space),
  + `3x` (half-width space),
  + `4x` (zero-width space),
  + `5x` (quarter-width space),
  + `6x` (1/8-width space),
  + `7x` (1/16-width space),
